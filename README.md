# Interactive 3D Developer Portfolio

An immersive, interactive developer portfolio powered by **Three.js** and **React Three Fiber**. Renders real-time 3D scenes — a floating computer, rotating tech-stack balls, and an interactive globe — with smooth Framer Motion animations.

🔗 **Live Demo:** https://sumit-portfolio-3d.netlify.app

## ✨ Features
- Real-time 3D scenes with React Three Fiber & Drei
- Interactive globe, floating desktop & rotating tech balls
- Smooth Framer Motion transitions and reveal animations
- Sections: hero, about, experience, tech, works, contact
- Fully responsive across devices

## 🛠️ Tech Stack
**React** · Three.js · React Three Fiber · Drei · Framer Motion · Tailwind CSS · Vite

## 🚀 Getting Started
```bash
npm install
npm run dev      # http://localhost:5173
```

## 📦 Build
```bash
npm run build
npm run preview
```
